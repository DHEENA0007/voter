Project Title

Secure Mobile Biometric Voting System

1. Project Overview

The Secure Mobile Biometric Voting System is a fully mobile-based election management application designed to conduct secure digital voting using mobile fingerprint authentication.

The system allows both Admin and Voters to use the same Flutter mobile application. Based on login credentials, users are redirected to their respective portals.

The application ensures:

Secure authentication using Voter ID + Passcode or Mobile Fingerprint.

One vote per voter per election.

No fingerprint data stored in the backend database.

Admin approval before voter activation.

Real-time election monitoring and automated result publishing.

2. Technology Stack

Backend: Python Django (REST API Architecture)
Frontend: Flutter (Single Mobile Application for Admin & Voter)
Database: SQLite3 (Upgradeable to PostgreSQL in production)
Biometric Authentication: Native Mobile Fingerprint Scanner (Android / iOS Biometric API)
Security: HTTPS + Encrypted Passcode Storage

3. System Architecture

The Flutter mobile application communicates securely with the Django backend using REST APIs.

Fingerprint authentication is handled entirely by the device’s built-in biometric system. The application does not store raw fingerprint data in the database. Instead, the mobile device verifies the fingerprint locally before granting access.

After authentication, the backend verifies voter approval and election eligibility.

5. Login System (Unified Mobile Login Screen)

The application contains a single login screen.

At the top, there is a secure toggle shield button:

User Login

Admin Login

Based on selection, the login method changes.

5.1 Admin Login

Admin logs in using:

Username

Password

After successful authentication, admin is redirected to the Admin Dashboard.

5.2 Voter Login

Voter has two login options:

Option 1: Voter ID + Passcode

Enter Voter ID

Enter Passcode

Backend verifies approval status

If approved → Redirect to Voter Portal

Option 2: Login with Fingerprint

Tap “Login with Fingerprint”

Mobile fingerprint scanner activates

Device verifies biometric authentication

Backend validates Voter ID

Redirect to Voter Portal

No face or image validation is used.

Fingerprint data is not stored in backend. Only mobile device biometric verification is used.
